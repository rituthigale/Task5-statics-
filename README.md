# Task5-statics-# 📊 Statistical Analysis Project

This project performs statistical analysis using Python.

## 🔧 Technologies Used
- Python
- Pandas
- NumPy
- SciPy
- Matplotlib

## 📌 Features
- Mean, Median, Mode Calculation
- P-value Calculation
- One Sample T-Test
- Data Visualization (Histogram)

## 📂 Project Structure
Statistical-Analysis-Project/
│── analysis.py  
│── data.csv  
│── report.txt  
│── README.md  
│── requirements.txt  

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python analysis.py

## 📊 Output
- Displays statistical results in console  
- Generates report.txt file  
- Shows histogram graph  

## 📌 Conclusion
This project demonstrates basic statistical analysis and hypothesis testing using Python.
